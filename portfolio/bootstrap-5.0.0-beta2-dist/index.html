About

Saya adalah seorang mahasiswa biasa yang ingin menjadi Web Developer. Saya termasuk orang yang sangat awam mengenai dunia IT, karena saya tertarik dengan dunia IT sekitar tahun 2020. Awalnya saya menyukai desain, tetapi setelah saya mempelajari Web Programming. Saya mulai tertarik, terutama di Front-End, karena sangat berhubungan dengan desain.

Saya adalah lulusan dari Sekolah Menengah Kejuruan di jurusan Teknik Instalasi Tenaga Listrik, saya tertarik dengan dunia IT bermula pada saat saya melihat di youtube tentang orang orang yang membuat websitenya sendiri, membuat gamenya sendiri dan hal lainnya. Dari situlah saya mulai tertarik dengan pemrograman dan memutuskan untuk mengambil jurusan Teknik Informatika di USM. Banyak bidang-bidang di dunia IT yang membuat saya tertarik, diantaranya yaitu

  • Web Developer
  • Game Developer
  • Software Developer
  • Network Architect
